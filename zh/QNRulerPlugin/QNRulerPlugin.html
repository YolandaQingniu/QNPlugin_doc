QNRulerPlugin

围度尺设备管理

API

setRulerPlugin

设置围度尺管理类

参数
名称 类型 说明
plugin QNPlugin 基础管理类
返回值

类型：int 详情见状态码列表

setDeviceListener

设置设备监听器

参数
名称 类型 说明
deviceListener QNRulerDeviceListener 设置设备监听器

setDataListener

设置设备数据监听器

参数
名称 类型 说明
dataListener QNRulerDataListener 设置数据监听器

connectDevice

连接设备

参数
名称 类型 说明
device QNRulerDevice 围度尺设备
返回值

类型：int 详情见状态码列表

cancelConnectDevice

断开设备连接

参数
名称 类型 说明
device QNRulerDevice 围度尺设备
