QNHeightWeightScalePlugin

身高体重秤管理类

  • 用于控制身高体重秤的使用；

覆盖方法

setScalePlugin

设置身高体重秤管理类

参数

名称 类型 说明
plugin QNPlugin 基础管理类
callback QNResultCallback 回调状态

setDeviceListener:

设置设备监听

参数

名称 类型 说明
deviceListener QNHeightWeightScaleDeviceListener 设备监听对象

setStatusListener:

设置设备状态监听

参数

名称 类型 说明
statusListener QNHeightWeightScaleStatusListener 设备监听对象

setDataListener:

设置设备数据监听

参数

名称 类型 说明
dataListener QNHeightWeightScaleDataListener 设备监听对象

connectHeightWeightScaleDevice

链接设备

参数

名称 类型 说明
device QNHeightWeightScaleDevice 设备对象
operate QNHeightWeightScaleOperate 可选控制信息
callback QNResultCallback 回调状态

cancelConnectHeightWeightScaleDevice:

断开链接

参数

名称 类型 说明
device QNHeightWeightScaleDevice 设备对象
