QNPlugin

SDK初始入口组件

主要功能：

  • 蓝牙设备基本管理
  • SDK授权信息验证
  • 系统蓝牙状态监听、扫描、连接

API

initSdk

根据appId进行SDK初始化，在初始化过程中会根据所使用的appId与配置文件进行可使用权限验证。

对于Android端建议在Application.onCreate中进行初始化； 对于iOS端建议在AppDelegate.didFinishLaunchingWithOptions中进行初始化。

注意： demo中所使用的123456789appid仅作为配合demo及调试使用，测试id具有不稳定性与随机变化性。接入方在正式使用时必须使用正式appId

参数
名称 类型 说明
appId String Yolanda提供的唯一标识
filePath String 配置文件的文件路径
callback QnResultCallback 回调方法，返回初始化的结果 int 详情见状态码列表

setScanListener

设置扫描状态监听器，主要用于监听蓝牙启动或停止扫描的动作监听。

参数
名称 类型 说明
listener QNScanListener 设置扫描状态监听

setSysBleStatusListener

设置系统蓝牙状态监听，只要系统蓝牙状态发生变化，即会回调该方法。

参数
名称 类型 说明
listener QNSysBleStatusListener 设置系统蓝牙状态变化监听

setLogListener

设置日志输出监听，用于APP记录SDK输出的日志信息。

参数
名称 类型 说明
listener QNLogListener 设置日志输出监听

getSysBleStatus

获取当前系统蓝牙状态

返回值

类型：QNSysBleStatus

startScan

启动蓝牙扫描

返回值

类型：int 详情见状态码列表

stopScan

停止蓝牙扫描
