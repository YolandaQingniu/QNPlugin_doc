QNBPMachineDataListener

QNBPMachineDataListener

Device data listener, used to monitor and call back device measurement data

API

onReceiveRealTimeData

Blood pressure monitor real-time measurement completion result data callback

parameter
name type description
measureResult QNBPMachineMeasureResult 本次测量结果状态
data QNBPMachineData Measurement data result, when the measurement fails, data is nil
device QNBPMachineDevice Device

onReceiveStoredData

Receive the stored data list callback

parameter
name type description
data QNBPMachineData Store data results
device QNBPMachineDevice Device

QNBPMachineMeasureResult

名称 含义
QNBPMachineMeasureResultUndone Not measured
QNBPMachineMeasureResultErr1 Device not calibrated
QNBPMachineMeasureResultErr2 Cuff disengagement/excessive movement
QNBPMachineMeasureResultErr3 High voltage detection failed
QNBPMachineMeasureResultErr4 Low voltage detection failed
QNBPMachineMeasureResultErr5 Blood pressure pulses not detected
QNBPMachineMeasureResultErr6 Measurement timeout
QNBPMachineMeasureResultErr7 overpressure
QNBPMachineMeasureResultErr8 Air leakage occurs during pressurization
QNBPMachineMeasureResultErr9 cuff not attached
QNBPMachineMeasureResultSuccess Measurement success
