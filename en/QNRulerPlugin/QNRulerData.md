# QNRulerData
围度尺测量数据

#### 成员变量
| 名称        | 类型                                                                    | 描述        | 
|-----------|-----------------------------------------------------------------------|-----------|
| value     | String                                                                | 测量长度      |
| unit      | [LengthEnum](./QNRulerData.md#LengthEnum) | 测量长度对应的单位 |
| timeStamp | String                                                                | 时间戳，单位s   |

#### LengthEnum
|名称  | 含义   |
| ------ |------|
|UNIT_CM  | cm   |
|UNIT_IN  | inch |
