QNScaleStatusListener

Device status listener, used to listen for callback device connection status

API

onConnectedSuccess

Home scale equipment connected successfully

parameter
name type description
device QNScaleDevice Household scale equipment object

onConnectFail

Home scale device connection failed

parameter
name type description
code int See list of status codes for details code
device QNScaleDevice Household scale equipment object

onReadyInteractResult

The home scale equipment is ready to start setting up the interaction

parameter
name type description
code int See list of status codes for details code
device QNScaleDevice Household scale equipment object

onDisconnected

The household scale device is disconnected

parameter
name type description
device QNScaleDevice Household scale equipment object
