API

setScalePlugin

Set up household scale management class

parameter
name type description
plugin QNPlugin Basic management
Return value

Type：int See list of status codes for details

setDeviceListener

Set the device listener

parameter
name type description
deviceListener QNScaleDeviceListener Device status listener

setStatusListener

Set the device status monitor

parameter
name type description
statusListener QNScaleStatusListener Device status listener

setDataListener

Set the device data listener

parameter
name type description
dataListener QNScaleDataListener Device data listener

connectDevice

Connecting device

parameter
name type description
device QNScaleDevice Household weighing equipment
operate QNScaleOperate Setting item
Return value

Type：int See list of status codes for details

cancelConnectDevice

Disconnect the device

parameter
name type description
device QNScaleDevice Household weighing equipment

setDeviceUnit

Setup unit

parameter
name type description
device QNScaleDevice Household weighing equipment
unit QNUnit unit
Return value

Type：int See list of status codes for details

setMeasureUser

Set measurement user information

Note: During the measurement, if the measurement user information is not set in time. At this time, the result of receiving measurement data is incomplete, and it is necessary to call the completion data method to obtain the complete data.

parameter
name type description
device QNScaleDevice Household weighing equipment
user QNUser User information
Return value

Type：int See list of status codes for details
